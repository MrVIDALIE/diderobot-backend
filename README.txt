# DideRobot - Backend sécurisé

## 📦 Installation
1. Assurez-vous d'avoir Node.js installé.
2. Dans un terminal :
   ```bash
   npm install
   npm start
   ```

## 🔐 Configuration
Ajoutez un fichier `.env` avec votre clé API OpenAI :

```
OPENAI_API_KEY=sk-votrecleici
```

## 🌐 Déploiement
Déployez sur https://render.com ou autre plateforme Node.js.

## ⚠️ Sécurité
Ne mettez jamais votre clé API directement dans le frontend.
